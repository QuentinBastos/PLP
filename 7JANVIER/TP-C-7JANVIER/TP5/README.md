# TP5 - Table des Symboles et Environnement

Ce TP finalise l'interpréteur en ajoutant une table des symboles pour la gestion des variables et des définitions.

# Bibliothèques
* `stdio.h`, `stdlib.h`, `string.h`, `math.h`
* `stdbool.h`

# Références
* Cours sur la compilation et les tables de symboles.

# Difficulté
* Gestion de la portée des variables (si applicable).
* Association nom-valeur efficace.
* Gestion de la mémoire pour les symboles.

# Commentaires
* Le module `symbols.c` gère l'ajout et la recherche de variables.
* L'interpréteur reconnaît maintenant des identificateurs définis par l'utilisateur.
* Compilation : `make`
* Exécution : `./repl`
