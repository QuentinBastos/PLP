# TP4 - Gestion de Pile et Évaluation Avancée

Ce TP étend les fonctionnalités du précédent en introduisant une structure de pile pour gérer des expressions plus complexes ou des contextes d'exécution.

# Bibliothèques
* `stdio.h`, `stdlib.h`, `string.h`, `math.h`
* `stdbool.h` : Pour le type booléen.

# Références
* Cours sur les structures de données (Piles/Files).

# Difficulté
* Implémentation générique ou spécifique de la pile (`pile.c`).
* Gestion de la mémoire (push/pop et libération).

# Commentaires
* Le module `pile.c` permet de stocker des résultats intermédiaires ou des contextes.
* Compilation : `make`
* Exécution : `./repl`
