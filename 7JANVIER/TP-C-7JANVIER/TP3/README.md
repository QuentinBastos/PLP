# TP3 - Analyseur Syntaxique et Évaluateur

Ce TP implémente un analyseur syntaxique et un évaluateur pour une calculatrice ou un interpréteur simple en C.

# Bibliothèques
* `stdio.h` : Entrées/sorties standard.
* `stdlib.h` : Gestion de la mémoire et utilitaires.
* `string.h` : Manipulation de chaînes de caractères.
* `math.h` : Fonctions mathématiques (pour l'évaluation).

# Références
* Cours de Programmation et Langages.

# Difficulté
* Gestion de l'analyse lexicale (lexer).
* Construction de l'arbre syntaxique (parser).
* Évaluation récursive.

# Commentaires
* Le programme utilise un REPL (Read-Eval-Print Loop) pour interagir avec l'utilisateur.
* Compilation : `make`
* Exécution : `./repl`
